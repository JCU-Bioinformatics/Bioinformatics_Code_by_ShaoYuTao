iPromoter-5mC Server User Guide

Descript

This prediction website is based on known human small cell lung cancer cell coefficients based on sites with significant promoter methylation levels. The main goal is to achieve the prediction of all 5mC promoters in human small cell lung cancer cell lines.

Intput file

Users can submit a multi-sequence file (in FASTA format)of submit sequence information by clicking a button.The final prediction model will send the predicted results to the mailbox submitted by the user through the mailbox.

Program name

Users can choose the appropriate name to identify the prediction result files

Content description

"Pre_lable" means the predicted label category (whether it's 5mC or not); "Negative_probably" means the probability value predicted for a non-5mc site; "Positive_probably" represents the probability value predicted for the 5mC site.
